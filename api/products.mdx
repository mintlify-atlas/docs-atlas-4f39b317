sidebarTitle: "Products"

---

The Products API manages the Yakult catalog — the source of truth for what can be sold, at what price, and how much remains in stock. Each product is identified by a human-readable SKU that must be unique across the catalog. When an order is placed, stock is deducted automatically; use these endpoints to inspect or correct inventory levels at any time. No authentication is required on any products endpoint.

---

### GET /api/productos

Returns the full product catalog ordered alphabetically by name, including current stock levels for every item. Products with a blank category are returned with `"General"` as the category value.

**Example request**

```bash
curl http://localhost:3000/api/productos
```

**Example response**

```json
[
  {
    "id": 1,
    "nombre": "Yakult Light 65ml",
    "sku": "YKL-LITE-65",
    "precio": 16.50,
    "stock": 300,
    "categoria": "General",
    "creado_en": "2025-01-01T00:00:00.000Z"
  },
  {
    "id": 2,
    "nombre": "Yakult Original 65ml",
    "sku": "YKL-ORIG-65",
    "precio": 15.00,
    "stock": 500,
    "categoria": "Bebida probiótica",
    "creado_en": "2025-01-01T00:00:00.000Z"
  }
]
```

<ResponseField name="id" type="number">Unique numeric ID of the product.</ResponseField>
<ResponseField name="nombre" type="string">Display name of the product.</ResponseField>
<ResponseField name="sku" type="string">Unique stock-keeping unit identifier.</ResponseField>
<ResponseField name="precio" type="number">Unit price in the local currency.</ResponseField>
<ResponseField name="stock" type="number">Available units currently in inventory.</ResponseField>
<ResponseField name="categoria" type="string">Product category label. Blank categories are returned as `"General"`.</ResponseField>
<ResponseField name="creado_en" type="string">ISO 8601 timestamp of when the product was added.</ResponseField>

### POST /api/productos

Adds a new product to the catalog. No authentication is required.

<ParamField body="nombre" type="string" required>
  Display name of the product, e.g. `"Yakult Original 65ml"`.
</ParamField>

<ParamField body="sku" type="string" required>
  Unique stock-keeping unit code, e.g. `"YKL-ORIG-65"`.
</ParamField>

<ParamField body="precio" type="number" required>
  Unit sale price in the local currency.
</ParamField>

<ParamField body="stock" type="number" required>
  Initial inventory quantity.
</ParamField>

<ParamField body="categoria" type="string">
  Optional category label to group products. Defaults to `"General"` if omitted or left blank.
</ParamField>

**Example request**

```bash
curl -X POST http://localhost:3000/api/productos \
  -H "Content-Type: application/json" \
  -d '{
    "nombre": "Yakult Original 65ml",
    "sku": "YKL-ORIG-65",
    "precio": 15.00,
    "stock": 500,
    "categoria": "Bebida probiótica"
  }'
```

**Example response**

```json
{
  "id": 1,
  "nombre": "Yakult Original 65ml",
  "sku": "YKL-ORIG-65",
  "precio": 15.00,
  "stock": 500,
  "categoria": "Bebida probiótica"
}
```

<ResponseField name="id" type="number">Auto-generated ID assigned to the new product.</ResponseField>
<ResponseField name="nombre" type="string">Product name as stored.</ResponseField>
<ResponseField name="sku" type="string">SKU as stored.</ResponseField>
<ResponseField name="precio" type="number">Unit price as stored.</ResponseField>
<ResponseField name="stock" type="number">Initial stock as stored.</ResponseField>
<ResponseField name="categoria" type="string">Category label as stored. Defaults to `"General"` when not provided.</ResponseField>

### PUT /api/productos/:id

Replaces all editable fields on an existing product. Supply all five fields — `nombre`, `sku`, `precio`, `stock`, and `categoria`. No authentication is required.

<ParamField path="id" type="number" required>
  Numeric ID of the product to update.
</ParamField>

<ParamField body="nombre" type="string" required>
  Updated product name.
</ParamField>

<ParamField body="sku" type="string" required>
  Updated SKU.
</ParamField>

<ParamField body="precio" type="number" required>
  Updated unit price.
</ParamField>

<ParamField body="stock" type="number" required>
  Updated stock level. Use this to manually correct inventory discrepancies.
</ParamField>

<ParamField body="categoria" type="string">
  Updated category label. Defaults to `"General"` if omitted or left blank.
</ParamField>

**Example request**

```bash
curl -X PUT http://localhost:3000/api/productos/1 \
  -H "Content-Type: application/json" \
  -d '{
    "nombre": "Yakult Original 65ml",
    "sku": "YKL-ORIG-65",
    "precio": 16.00,
    "stock": 480,
    "categoria": "Bebida probiótica"
  }'
```

**Example response**

```json
{ "ok": true }
```

---

### DELETE /api/productos/:id

Permanently removes a product from the catalog. This action cannot be undone. No authentication is required.

<ParamField path="id" type="number" required>
  Numeric ID of the product to delete.
</ParamField>

**Example request**

```bash
curl -X DELETE http://localhost:3000/api/productos/1
```

**Example response**

```json
{ "ok": true }
```
